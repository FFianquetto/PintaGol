Redes sociales (Firebase + Meta)

1) Firebase
   - Crea un proyecto en https://console.firebase.google.com
   - Authentication: habilita "Correo/contraseña" y "Facebook"
   - Para Facebook en Firebase, necesitas un App ID de Meta (mismo que en el paso 2)
   - Pega la configuración web en config/firebase.config.js

2) Facebook / Meta (mismo ID que uses en Firebase para "Facebook" en Auth)
   - https://developers.facebook.com → Crear app → p. ej. consumidor
   - Producto: Inicio de sesión con Facebook. URI de redirección OAuth válida:
     http://localhost:3333/oauth/meta/callback
     (Si abres con 127.0.0.1:3333, añade también http://127.0.0.1:3333/oauth/meta/callback. Ajusta el puerto a PORT o env.)
   - Crea "redes-secrets.json" en la raíz (plantilla: redes-secrets.example.json):
     { "metaAppId": "…", "metaAppSecret": "…" }

3) Instagram
   - El botón "Instagram" usa inicio con Facebook/Graph: obtenemos el email de
     Meta y lo cotejamos con el correo con el que iniciaste en Pinta Gol. No
     incluimos Twitter, como acordado.

4) Inicio
   - npm start
   - En el lobby: "Redes sociales" → registro/inicio con correo → vincular Facebook/Instagram
