# Efectos de sonido de items

Esta carpeta concentra los efectos de sonido para items del juego.

## Convencion sugerida de archivos

- `item_pickup_coin.wav`
- `item_pickup_health.wav`
- `item_pickup_shield.wav`
- `item_spawn.wav`
- `item_expire.wav`

## Notas

- Formato recomendado: `wav` u `ogg`.
- Mantener volumen normalizado para evitar saltos bruscos.
- Evitar nombres genericos como `sound1.wav`.
